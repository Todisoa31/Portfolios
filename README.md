# 💼 Portfolio - Todisoa Herinajanahay

Bienvenue sur mon portfolio de développeur Full Stack JavaScript.

Ce projet présente mon parcours professionnel, mes réalisations, mes compétences techniques ainsi que plusieurs études de cas mettant en avant les défis techniques que j'ai rencontrés et les solutions mises en œuvre.

Le portfolio a été conçu comme une véritable vitrine technique, avec un design moderne, des animations fluides et une expérience utilisateur soignée.
